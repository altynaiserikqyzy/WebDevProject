# NoRetake KBTU

## Project Description
NoRetake KBTU is a student support platform where KBTU students can find help in different subjects from senior students or other students who understand the material well.

Tutors can publish their tutoring services, set the subject, price, and format of lessons. Students can search for tutors, contact them, arrange lessons, and communicate through chat. The platform also helps users keep track of planned study sessions.

The project is built with Angular and Django REST Framework.

## Main Features
- Authentication and authorization
- Tutor profiles and service listings
- Booking study sessions
- Chat between tutor and student
- Session scheduling
- User profile page
